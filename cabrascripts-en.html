PI Workflow

v0.9 Beta

PI Workflow is a comprehensive astrophotography processing interface that unifies the entire post-processing pipeline into a single environment. It organizes the processing steps dynamically while eliminating the clutter of dozens of temporary preview windows. It is designed to provide a smooth, structured transition from linear pre-processing to nonlinear stretching, star splitting, and final channel combination.

Universal Shared Preview Canvas

A single, high-performance, interactive preview panel shared across all tabs and tools. No more workspace clutter. Slider adjustments update in real-time via downsampling, with full-resolution calculations occurring only when you commit changes.

Interactive Split-Screen (Split View)

Divide the preview canvas with a vertical slider to inspect and compare "before" (original source) and "after" (processed candidate) details in real-time.

Multi-Algorithm Comparison Grid

Run and compare multiple tools simultaneously (e.g. ABE, DBE, and GraXpert for gradients; or BlurXTerminator and LHE for sharpening) in an integrated comparison grid.

8-Slot Transient Memory

Save up to 8 processing stages per tab and toggle between them instantly with simple left/right clicks to evaluate subtle parameter adjustments.

Installation

View GitHub Repository Detailed Instructions

Quick Installation Steps

  • 1
    Open PixInsight and navigate to Resources > Updates > Manage Repositories.
  • 2
    Click Add and paste the repository URL shown above.
  • 3
    Click OK, then select Resources > Updates > Check for Updates.
  • 4
    PixInsight will find the script package. Click Apply and restart PixInsight. It will appear under Script > Utilities > PI Workflow!
